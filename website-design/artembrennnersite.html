Welcome to My Website

This is a basic web layout that uses div containers to organize content. Divs are like boxes where you can set text and images, and allows you to control how things are positioned on the page and how they appear. CSS is used to format their appearance.

When you set up the navigation you always want to put it inside unordered list (ul) tags and/or list (li) tags. This makes it easy to format how your navigation appears.
